## ✨ Key Features

- **🎨 6 Bespoke Themes:** Includes a custom-built theme switcher with 6 palettes (*Dark Gold, Light Ivory, Ocean Blue, Forest Green, Midnight Purple, and Rose Pink*). Themes are automatically saved in `localStorage`.
- **🏘️ Dynamic Property Grid:** Users can dynamically filter premium properties by region without page reloads.
- **📲 Interactive Property Drawer:** Clicking on any property seamlessly slides in an elegant off-canvas drawer featuring image galleries, key statistics, full descriptions, and agent details.
- **📱 Fully Responsive Design:** Fluid layouts that adapt perfectly from ultra-wide displays to mobile devices, complete with a custom animated hamburger menu.
- **✨ Scroll Animations:** Utilizing the `IntersectionObserver` API for smooth, performance-friendly scroll-reveal animations across all sections.

---

## 🛠️ Technical Highlights
- **Zero Dependencies:** Built entirely using Vanilla JavaScript, making it incredibly fast and lightweight.
- **CSS Variables Engine:** The entire theme switching mechanism operates using a complex CSS `:root` variable structure.
- **Dynamic Content Rendering:** Property cards and the off-canvas drawer are populated dynamically by a JavaScript array object, allowing easy database integration in the future.

---

## 🤝 Usage & Customization
To customize the property listings, simply navigate to the `const PROPS = [...]` and `const EXTRA = {...}` arrays inside the `<script>` tag at the bottom of the [index.html](cci:7://file:///c:/Users/User2/Downloads/S.S/index.html:0:0-0:0) file. You can easily add, edit, or remove real estate listings, change map locations, and modify agent details.
